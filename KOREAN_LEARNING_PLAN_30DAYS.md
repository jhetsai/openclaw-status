## 🎬 第四關：韓劇學台詞（Day 24-30）

### 学习内容
- 看韓劇片段學習日常會話
- 模仿發音和語調
- 學習流行語
- Shadowing（跟讀法）練習

### 每日任务

| 日期 | 任務 | 練習時間 |
|------|------|---------|
| Day 24 | 選擇1集喜歡的韓劇 | 50分鐘 |
| Day 25 | 韓劇第一遍（看中文字幕）| 50分鐘 |
| Day 26 | 韓劇第二遍（開韓文字幕）| 50分鐘 |
| Day 27 | 跟讀經典台詞 | 50分鐘 |
| Day 28 | 抄寫並背誦喜歡的台詞 | 50分鐘 |
| Day 29 | 用該台詞造句 | 50分鐘 |
| Day 30 | 總複習＋未來學習計畫 | 50分鐘 |

### 推薦韓劇（適合學習）
| 戲劇 | 理由 |
|------|------|
| 淺草高校 | 日常生活對話，簡單易懂 |
| 繼承者們 | 校園劇，詞彙不難 |
| 鬼怪 | 句子優美，適合背誦 |
| 經常請吃飯的姐姐 | 現代日常會話 |

### Shadowing 跟讀法
1. 聽一句（暫停）
2. 跟著說一遍（模仿語調）
3. 再聽一遍（確認）
4. 繼續下一句

## 💡 學習技巧

1. **每天30分鐘**勝過一次讀5小時
2. **寫下來**：手寫幫助記憶
3. **說出口**：啞巴式學習沒用
4. **用韓文想**：不要先想中文再翻譯
5. **錯了也沒關係**：敢說比說對更重要
