### 今天學到的概念

#### 1. LLM Wiki（Karpathy 提出的知識庫模式）
- **核心**：不是每次答題都重新找資料，而是讓 LLM 持續維護一個可累積的 Wiki
- **三層**：Raw Sources（唯讀）→ Wiki（可改）→ Schema（規則）
- **流程**：Ingest（新知識寫入）→ Query（從Wiki回答）→ Lint（定期健康檢查）
- **價值**：知識持續累積、cross-reference、矛盾標記

#### 2. LLM-Wiki-Worker（實際專案）
- **技術**：Cloudflare Worker + Queue + Telegram + GitHub
- **規則驅動**：不是靠 prompt，是靠 AGENTS.md + rules/*.md
- **寫檔限制**：只能寫 wiki/ 路徑
- **參考價值**：可以簡化用在 OpenClaw 系統裡
