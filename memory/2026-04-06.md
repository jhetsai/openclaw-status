## 農業天氣頁面（最終版）

### 天氣頁面網址
https://pub-ad498842971c4801a54fabd88ffa4a7f.r2.dev/weather_agri.html

### 功能清單
| 功能 | 說明 |
|------|------|
| CWA 即時天氣 | 溫度、降雨、濕度、風速 |
| 每小時曲線圖 | Open-Meteo **從現在起的未來24小時**，溫度+降雨機率 |
| CWA 18小時預報 | 每12小時，天氣+溫度+降雨等級(圖示)+風速(獨立欄位) |
| Open-Meteo 7天預報 | 早/中/晚三時段 |
| 農務建議 | 根據溫度和降雨自動建議 |

### CWA 18小時預報 顯示欄位
- 時間（每12小時，3筆：今晚18時 / 明天06時 / 明天18時）
- 天氣圖示
- 天氣描述 + 溫度區間
- 降雨等級（🌧️🌧️高 / 🌧️中 / 🌦️低 / ☀️無）
- 風速（🌬️ km/h，獨立欄位）

### 每小時曲線圖 重要修正
- **從現在起的未來24小時**，跳過過去的資料
- 使用 `omDate >= now` 判斷過濾

### 降雨等級圖示
- 🌧️🌧️ 高（≥60%）
- 🌧️ 中（≥30%）
- 🌦️ 低（>10%）
- ☀️ 無（≤10%）

### 風速等級
- 強（≥30 km/h）
- 中（≥15 km/h）
- 輕（>5 km/h）
- 微（≤5 km/h）

### 技術細節
- **Chart.js v4.4.0** 繪製曲線圖
- **雙Y軸**：左軸溫度(°C)、右軸降雨機率(%)
- **CWA API Key**：CWA_API_KEY_REDACTED
- **風速來源**：Open-Meteo
