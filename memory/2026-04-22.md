## 重要教訓
- **時序問題**：React 中 useState 的 setter 在 data load 完成後執行，但若在定義陣列之前就呼叫，會造成空白畫面
- **npm run build 失敗**：每次 `npm run build` 都產生白畫面，需用 Python 直接修改 minified JS
- **圖表 `selectedStock=null` 問題**：首次載入時 `selectedStock` 為 null，需等 userSelection state 更新後才會顯示圖表
