## 美股行情 Tab ${:.2f} Bug（14:07 修復）

### 問題現象
gen-combined-tab.py 產出的 HTML，美股行情（AAPL/MSFT/BND）的現價和昨收都顯示為 `${:.2f}` 而不是實際數值。

### 根本原因
Python `str.format()` 和字串中的 `${:.2f` 產生衝突：
```python
# 錯誤示範
tmpl = "<td>${:.2f}</td><td>${:.2f}</td>"
result = tmpl.format(258.83, 259.20)
# 輸出：<td>${:.2f}</td><td>${:.2f}</td>  ← 沒被置換！
```

`format()` 把 `${` 視為另一個未填充的預留位置，導致整個 `${:.2f}` 被忽略。

### 修復方式
把 `$` 和數字分開處理：
```python
price_str = "$" + "{:.2f}".format(p)
prev_str  = "$" + "{:.2f}".format(prev)
row = "<tr>...<td>"+price_str+"</td><td>"+prev_str+"</td>...{:+.2f}%...".format(chg)
```

### 修改檔案
- `scripts/gen-combined-tab.py` 第 209-210 行（us_rt_mob / us_rt_desk）

### 驗證結果
```
AAPL  $258.83  $259.20  -0.14%
MSFT  $393.11  $384.37  +2.27%
BND   $73.98   $73.80   +0.24%
```

漲跌 % 也正確使用 `{:+.2f}` 顯示正負號。

gen-combined-tab.py 產出的 HTML，美股行情（AAPL/MSFT/BND）的現價和昨收都顯示為 `${:.2f}` 而不是實際數值。

### 根本原因
Python `str.format()` 和字串中的 `${:.2f` 產生衝突：
```python
# 錯誤示範
tmpl = "<td>${:.2f}</td><td>${:.2f}</td>"
result = tmpl.format(258.83, 259.20)
# 輸出：<td>${:.2f}</td><td>${:.2f}</td>  ← 沒被置換！
```

`format()` 把 `${` 視為另一個未填充的預留位置，導致整個 `${:.2f}` 被忽略。

### 修復方式
把 `$` 和數字分開處理：
```python
price_str = "$" + "{:.2f}".format(p)
prev_str  = "$" + "{:.2f}".format(prev)
row = "<tr>...<td>"+price_str+"</td><td>"+prev_str+"</td>...{:+.2f}%...".format(chg)
```

### 修改檔案
- `scripts/gen-combined-tab.py` 第 209-210 行（us_rt_mob / us_rt_desk）

### 驗證結果
```
AAPL  $258.83  $259.20  -0.14%
MSFT  $393.11  $384.37  +2.27%
BND   $73.98   $73.80   +0.24%
```

漲跌 % 也正確使用 `{:+.2f}` 顯示正負號。
