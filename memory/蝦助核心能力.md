## 1. 太陽能發電模型（已確認）

**地點：** 水林鄉，雲林（緯度 23.71°N，經度 120.29°E）

**發電公式 v4（UV分段）：**
```
UV < 3：
  發電 = min(85W, UV × 18.2 × 3.0 × cloud_factor × sin(仰角))
UV ≥ 3：
  發電 = min(85W, UV × 18.2 × cloud_factor × sin(仰角))

cloud_factor = 1 - (雲層% × 0.6)
```
