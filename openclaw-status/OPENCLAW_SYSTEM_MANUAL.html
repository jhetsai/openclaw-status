1. 備份方式

指令說明

cp -r ~/.openclaw/agents ~/backup_agents_$(date +%Y%m%d)複製代理設定

~/openclaw_backups/

2. 還原教學

✅ 推薦方式：使用還原腳本

~/openclaw/scripts/restore_openclaw.sh

手動還原

步驟指令
1. 停止服務openclaw gateway stop
2. 複製備份cp ~/backup_openclaw.json ~/.openclaw/openclaw.json
3. 重啟服務openclaw gateway start

3. 系統維護常用指令

指令用途
openclaw gateway status查看服務狀態
openclaw gateway restart重啟 Gateway
openclaw plugins list查看已安裝插件
openclaw pairing list line查看 LINE 配對請求
tail -f /tmp/openclaw/openclaw-*.log查看即時日誌
