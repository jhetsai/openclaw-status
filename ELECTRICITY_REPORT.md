## 4.1 頁面佈局

```

│  │              詳細資料表格                            │   │
│  │  電號 | 期別 | 年度 | 用電(度) | 電費(元)          │   │
│  │  ────────────────────────────────────────────────  │   │
│  │                                                    │   │
│  └────────────────────────────────────────────────────┘   │
└────────────────────────────────────────────────────────────┘
```

## 4.2 篩選器規格

### S1：電號選擇器
- 選項：all（全選）+ 14 組電號
- 功能：依電號過濾資料

### S2：年度選擇器
- 選項：all（全選）+ 各年度
- 功能：依年份過濾資料

### S3：模式切換器
- 選項：y（依年度）/ q（依期別）/ p（指定期別）
- 功能：切換顯示模式

### S4：期別選擇器（p-mode 專用）
- 選項：各期別代碼
- 功能：在 p-mode 下篩選特定期別

## 4.3 視圖模式規格

### y-mode（依年度）
- 合併同一年度所有期別
- X軸：年度
- Y軸：用電度數或電費金額
- 顯示：柱狀圖 + 年度總計表

### q-mode（依期別每2月）
- 顯示每期別的用電/電費
- X軸：期別代碼
- Y軸：用電度數或電費金額
- 顯示：柱狀圖 + 期別資料表

### p-mode（指定期別各電號）
- 隱藏圖表，只顯示表格
- 顯示該期各電號的用電比較
- 適用：跨電號分析

## 4.4 圖表規格

| 項目 | 規格 |
|------|------|
| 圖表類型 | SVG 柱狀圖 |
| 柱子寬度 | 動態計算（視窗寬度 / 資料筆數） |
| 最大高度 | 400px |
| 柱子顏色 | 用電=#3498db / 電費=#e74c3c |
| hover 效果 | 顯示數值 tooltip |

## 9.2 備份策略

1. **GitHub Repository**：所有版本都有完整歷史
2. **本地備份**：建議每季備份一次 HTML 檔案
3. **Excel 原始檔**：保留台電提供的原始資料

## 9.3 異常處理

| 異常情況 | 可能原因 | 處理方式 |
|----------|----------|----------|
| 某期無資料 | 新電號或資料未更新 | 檢查台電帳單 |
| 用電為 0 | 電表故障或暫停用電 | 聯繫台電確認 |
| 電費異常高 | 用電增加或計費錯誤 | 核對帳單詳情 |
