## 🚗 交通方式

### 去程：8月14日
```
水林 → 嘉義高鐵 → 台中高鐵 → 156公車 → 清泉崗機場 → 釜山
出發時間：早上（建議10:00前抵達清泉崗）
飛行時間：16:00 - 19:20（約2小時20分）
時差：+1小時（抵達當地約20:20）
```

### 回程：8月18日
```
釜山 → 清泉崗 → 156公車 → 台中高鐵 → 嘉義高鐵 → 水林
航班：13:35起飛 → 15:00抵達台灣
```

### 釜山當地交通
| 交通方式 | 建議場景 | 費用參考 |
|---------|---------|---------|
| **地鐵** | 各大景點 | ₩1,350~₩2,000/趟 |
| **公車** | 地鐵到不了的地方 | ₩1,200~₩1,800/趟 |
| **計程車** | 深夜或赶时间 | ₩3,800起跳 |
| **Uber** | 可綁卡直接扣款 | 方便推薦 |

> 💡 **建議購買**：「釜山通行證 Busan Pass」或 T-money 卡（類似悠遊卡，可儲值）

## 🏨 住宿

### 東橫INN 釜山站1號店
- **地址**：重慶南路地下88（重慶橋旁，釜山站1號出口步行3分鐘）
- **類型**：商務酒店，乾淨便宜
- **優點**：車站旁邊、來往機場方便、含早餐
- **預估費用**：₩50,000~₩70,000/晚（雙人房約₩80,000）
